# Coastline Extraction in Jiangsu Province (2018–2025)

This repository contains code for coastline extraction using multi-spectral indices (NDWI, MNDWI, NDVI) and MSIC method.

## Data
- Sentinel-2 and Landsat-8 imagery from Google Earth Engine

## Method
- Multi-spectral index selection
- Annual maximum spectral index compositing (MSIC)
- Transect-based coastline change analysis
